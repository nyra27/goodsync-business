## 块级传输这个细节，值得单独说一下

传统备份工具基本是这样的逻辑：文件有变化 → 重新传整个文件。一个 1GB 的数据库文件，你改了 10KB，它把 1GB 全传一遍。

GoodSync 的块级传输只传那 10KB。

这对服务器场景的意义特别直接：带宽省了，时间省了，备份窗口缩短了。如果你的业务涉及大文件频繁修改（比如数据库、视频素材、大型文档），这个功能长期用下来差距会很明显。
